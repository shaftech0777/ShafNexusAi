# Shaf Nexus AI 🚀

An advanced, full-stack AI-powered software engineering workspace and DevOps manager built with React, Vite, Express, and Google Gemini.

## Features ✨

- 💻 **Interactive Code Editor:** Edit workspace source files in real-time with integrated line numbering, auto-formatting, and diagnostic monitors.
- 🤖 **AI Assistant (Gemini API):** An intelligent copilot that helps write code, debug issues, and draft architectural plans.
- 🗄️ **Relational Database Management:** Interact with relational backends (including Postgres and virtual fallback layers) with real-time SQL console syntax outputs.
- 🌿 **Git & GitHub Workspace Controls:** Clone repositories, create commit hashes, and sync files to GitHub repositories using access tokens.
- 📱 **Native Android & Flutter Compiler Studio:** Configure mobile app configurations, compile APB/AAB visual packages with virtual Gradle assembly indicators, and simulate splash-screen splash transitions.
- 📊 **DevOps Metrics Indicator:** Track real-time container CPU loads, server latency metrics, database records, and active sync states.

## Tech Stack 🛠️

- **Frontend:** React + TypeScript + Tailwind CSS
- **Design Core:** Lucide Icons, Framer Motion transitions
- **Server:** Node.js + Express
- **AI Core:** Google Gemini SDK (`@google/genai`)

## Quick Start 🚀

1. Clone or download this project workspace.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your environment variables in `.env` (use `.env.example` as a template):
   ```env
   GEMINI_API_KEY=your-gemini-key
   PORT=3000
   ```
4. Start development mode:
   ```bash
   npm run dev
   ```
